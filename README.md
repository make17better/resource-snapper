# Resource Snapper

A resource snap service powered by NestJS.

## Quick Start

### Prerequisites

- node@20.17.0
- npm@10.9.0

### Installation

```bash
npm install
```

### Running the app

```bash
# development
npm run start

# watch mode
npm run start:dev

# production mode
npm run start:prod
```

## API Documentation

By default, the API documentation is available at:

- Swagger UI: `http://localhost:3000/docs`
- OpenAPI Spec: `http://localhost:3000/docs-json`
